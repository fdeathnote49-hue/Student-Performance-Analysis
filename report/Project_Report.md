# Student Performance Analysis

## 1. Introduction

Student Performance Analysis is a Data Analytics project developed using Excel, SQL, Python, and Power BI. The objective of this project is to analyze student academic performance and generate meaningful insights through data analysis and visualization.

---

## 2. Objectives

- Analyze student performance
- Find top-performing students
- Analyze attendance
- Department-wise comparison
- Grade distribution analysis
- Create interactive dashboard

---

## 3. Tools Used

- Microsoft Excel
- MySQL
- Python
- Power BI
- VS Code

---

## 4. Dataset

The dataset contains information about 20 students.

Fields included:

- Student ID
- Student Name
- Gender
- Department
- Attendance
- Total Marks
- Percentage
- Grade

---

## 5. SQL Analysis

The following SQL operations were performed:

- SELECT
- WHERE
- ORDER BY
- GROUP BY
- COUNT
- AVG
- MAX
- MIN
- SUM
- LIKE

---

## 6. Python Analysis

Python libraries used:

- pandas
- matplotlib
- openpyxl

Operations performed:

- Data Import
- Data Cleaning
- Exploratory Data Analysis
- Visualization

---

## 7. Charts Created

- Department-wise Average Percentage
- Grade Distribution
- Attendance Analysis
- Percentage Histogram

---

## 8. Conclusion

This project demonstrates how data analysis techniques can be used to evaluate student performance and support better academic decision-making.
