# 🎓 Student Performance Analysis

## 📌 Project Overview

This project analyzes student academic performance using Excel, SQL, Python and Power BI.

The objective is to identify performance trends, department-wise analysis, grade distribution, attendance impact and overall student insights.

---

## 🛠 Technologies Used

- Microsoft Excel
- SQL
- Python
- Pandas
- Matplotlib
- Power BI
